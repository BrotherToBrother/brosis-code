🧠 Super Humanizer - Full Setup and Usage Guide

Welcome! This guide will show you exactly how to install, prepare, and use the Super Humanizer tool — step-by-step — no experience needed!

🛠 Step 1: Install Ollama

  1. Visit the Ollama website and download the installer for your system (Windows, Mac, or Linux).
  2. Install Ollama following the wizard (Next, Next, Finish).
  3. After installing, make sure Ollama is running!

📦 Step 2: Install a Model

Ollama uses "models" — brain files for AI. You must install a model once before using Super Humanizer.

  1. Open your Command Prompt (CMD) or Terminal.
  2. Type a command like:
ollama run llama3
  • For Academic Writing: Use llama3 — best for serious, human-sounding academic tone.
  • For Friendly Writing: Try mistral or orca-mini.

Tip: Pulling a model only happens once. After that, it stays installed!

🖥 Step 3: Install Super Humanizer

  1. Go to the Releases page and download Super_Humanizer_Setup.exe.
  2. Run the installer and follow the setup wizard (Next, Install, Finish).

🚀 Step 4: How to Use Super Humanizer

  1. Launch the app from your Desktop shortcut.
  2. Select your Input folder (your original .docx files).
  3. Select your Output folder (where humanized files will be saved).
  4. Pick your AI model from the dropdown (must be installed in Ollama).
  5. (Optional) Enable Grammar Correction if needed.
  6. Click the big 🚀 Start Humanizing button and watch it process automatically!

💡 Tips for Best Results

  • Use llama3 for professional, academic or research-style text.
  • Use mistral for friendly, conversational text.
  • Use orca-mini for lightweight or quick summaries.
  • Grammar Correction is optional — use it if you want polished English output!

⚙️ Requirements

  • ✅ Ollama installed and running.
  • ✅ At least one model pulled (like llama3, mistral, or orca-mini).
  • ✅ Python not needed if using the .exe version (only needed for source code).
  • ✅ Internet connection (only needed once for downloading a model).
